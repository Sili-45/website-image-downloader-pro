## Introduction

Website Image Downloader Pro is a fast and efficient tool that extracts and downloads images from webpages. With features like SVG to PNG conversion and image zipping, it makes managing and storing images a breeze. Ideal for marketers, researchers, and developers, this scraper offers a hassle-free image extraction solution.

### Key Features
- **Image URL Extraction**: Extracts all image URLs from provided web pages.
- **SVG to PNG Conversion**: Automatically converts SVG images to PNG format.
- **Image Downloading**: Downloads all images from extracted URLs.
- **Zipping Images**: Compresses all downloaded images into a single zip file for easy storage and transfer.

## Features

| Feature | Description |
|---------|-------------|

## What Data This Scraper Extracts

| Field Name | Field Description |
|------------|-------------------|
| image_url | The URL of the image extracted from the webpage. |
| input_url | The URL of the webpage where the image was found. |
| filename  | The name of the image file for easy reference. |
